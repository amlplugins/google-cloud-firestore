# @amlplugins/google-cloud-firestore

Native `@google-cloud/firestore` SDK re-exported under the `@amlplugins` namespace.

> Cloud Firestore.

## Install

```bash
npm install @amlplugins/google-cloud-firestore
```

## Use

```ts
import { /* named exports from @google-cloud/firestore */ } from "@amlplugins/google-cloud-firestore";
// or
import sdk from "@amlplugins/google-cloud-firestore";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
